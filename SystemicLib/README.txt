# Summary
#########################################################################################################

The Library has a handful of utility classes, mainly around dealing with Strings, dates and files. Most 
of the time they encapsulate functions that are available in some other libraries but are spread throughout 
them or have a limited functionality. With this library some additional functionality is added to some 
utility classes or functionality is wrapped in a more user-friendly way.

# Download Instructions

How to download this project:

Option 1 - As a Zip.
====================
Click on the button marked "ZIP" available from the Code tab.

Option 2 - Using a Git client.
==============================
From the command-line type: git clone git://github.com/SystemicPtyLtd/SystemicUtilityLibrary-java.git

Note that if you want to use this option but don't have the client installed, it can be 
downloaded from http://git-scm.com/download.
